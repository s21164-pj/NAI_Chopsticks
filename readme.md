## Sterowanie w grze

### Dostępne komendy:

- ('atak', x, y, z)
    - x - ręka atakująca
    - y - ręka atakowana
    - z - liczba palców które atakują
- ('rozdziel', x, y, z)
    - x - ręka z której rozdzielamy
    - y - ręka do której rozdzielamy
    - z - liczba palców które transferujemy
- Komenda pomocnicza "show moves"
